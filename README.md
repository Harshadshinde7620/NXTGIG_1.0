## \uc0\u55357 \u56534  Overview\
The NXTGIG Automation Framework is a Behaviour-Driven Development (BDD) based test automation framework designed for end-to-end web application testing.  \
It\'92s built using Java (17), Selenium WebDriver, Cucumber, TestNG, and Maven, following industry-standard practices such as Page Object Model (POM) and modular test design.\
\
This project demonstrates how BDD can be integrated into a scalable automation architecture suitable for real-world projects.\
\
---\
\
## \uc0\u9881 \u65039  Tech Stack\
- Language: Java 17  \
- Automation Tool: Selenium WebDriver  \
- BDD Framework: Cucumber (Gherkin)  \
- Test Runner: TestNG  \
- Build Tool: Maven  \
- Design Pattern: Page Object Model (POM)  \
- Reporting: TestNG Reports / Extent Reports  \
- Configuration: `config.properties`\
